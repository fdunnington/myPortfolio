Custom Bootstrap

A project to improve the design, functionality, and feature scope of an existing custom-built Bootstrap gaming platform

Work With Me

The Project

I joined this project after the initial site build as part of the design team.

Focal points of the project included:

  • understanding and improving user flow using site analytics, site mapping, and user stories

  • accessiblity (WCAG3)

  • site-wide design cohesivity

  • implementation of new-features

The Process

Research & analysis: Using site analytics ( Hotjar & Google Analytics) we determined the use-patterns of site users, and from this built flow diagrams to create a clear picture of the flow of users through the site.

With an established user flow for each key objective we used TOL tests to take a more in-depth look at any perceived areas of friction to uncover opportunites to improve, and to learn about the preferences of the users.

Design: The feedback collected enabled us to design the site structure according to identified use-patterns, and to optimize the UI for the desired outcomes.

Using wireframes and mockups we fine-tuned UI, and then used Figma-built interactive prototypes to test iterations and finalize designs pre-build.

Build: Once designs are finalised we converted the designs to stand-alone Bootstrap modules that were ready to pass on to the the dev team for integration with the site.
